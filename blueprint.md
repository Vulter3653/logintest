# SKKU Marketing Coffee Chat 블루프린트

## 개요
성균관대학교 마케팅 학우들을 위한 **실시간 커피 챗(Coffee Chat)** 플랫폼입니다. 별도의 파일 저장소(Storage) 없이 텍스트와 API 연동만으로 가볍고 빠르게 작동하며, 학우들 간의 지식 공유와 실시간 소통을 지원합니다.

## 브랜드 및 디자인 가이드
- **컬러 팔레트**: Evergreen (성균 그린) & Ginkgo Yellow (은행잎 노란색) 베이스의 다크/라이트 테마 지원.
- **UI 컨셉**: 부드러운 카드 레이아웃과 직관적인 모달 인증 시스템.

## 주요 기능 (저장소 미사용 방식)
- **실시간 글로벌 피드**: Firestore를 이용한 텍스트 기반 실시간 대화 및 답글(대댓글).
- **DiceBear 아바타 시스템**: 서버 저장 공간을 차지하지 않는 API 방식의 무제한 아바타 커스터마이징.
- **인증 및 프로필**: 닉네임 일괄 업데이트 기능이 포함된 이메일 인증 기반 회원 시스템.
- **댓글 반응**: 하트(좋아요) 기능을 통한 실시간 상호작용.
