# 로그인 화면 및 실시간 커뮤니티 블루프린트

## 개요
이 프로젝트는 사용자 인증 기능을 포함한 **실시간 글로벌 댓글 피드** 서비스를 제공합니다. 이메일/비밀번호 인증과 구글 소셜 로그인을 지원하며, 모든 사용자가 실시간으로 소통할 수 있는 현대적인 웹 애플리케이션입니다.

## 주요 기능 및 디자인
- **Firebase Authentication 통합**:
  - **이메일/비밀번호**: 기존 방식의 로그인 및 회원가입.
  - **구글 로그인 (Google OAuth)**: `signInWithPopup`을 이용한 간편 소셜 로그인 추가.
  - **비밀번호 재설정**: 가입된 이메일로 재설정 링크 발송.
- **실시간 글로벌 피드 (`<comments-section>`)**:
  - **Firestore 연동**: 모든 사용자가 하나의 게시판을 실시간으로 공유.
  - **사용자 식별**: 이메일 또는 구글 프로필 이름을 작성자로 표시.
  - **나의 댓글 강조**: 본인이 쓴 글은 별도의 스타일과 배지로 시각적 구분.
- **현대적인 UI/UX**:
  - **구글 버튼**: 브랜드 가이드를 준수하는 세련된 구글 로그인 버튼 디자인.
  - **다크 테마**: OKLCH 기반의 세련된 다크 모드 UI.
  - **반응형 최적화**: 모든 기기에서 최적의 레이아웃 제공.
