# SKKU Marketing Coffee Chat 블루프린트

## 개요
성균관대학교 마케팅 학우들을 위한 **실시간 커피 챗(Coffee Chat)** 플랫폼입니다. 강력한 이메일 인증 시스템을 통해 신뢰할 수 있는 커뮤니티 환경을 제공하며, 실시간 소통과 프로필 관리 기능을 지원합니다.

## 주요 기능 및 보안 정책
- **통합 인증 시스템**:
  - **이메일 인증 (Email Verification)**: 회원가입 시 인증 메일 발송 필수. 인증 완료 전까지는 읽기 권한만 부여하고 쓰기 권한은 제한하여 보안 강화.
  - **구글 로그인**: 즉각적인 인증 및 연동 가입 지원.
  - **비밀번호 재설정**: 가입된 이메일을 통한 안전한 복구 로직.
- **프로필 관리**:
  - **닉네임 기반 소통**: 이메일 대신 고유한 닉네임으로 피드 활동.
  - **개인 설정 페이지**: 언제든지 본인의 닉네임을 수정하고 계정 상태 확인 가능.
- **실시간 커뮤니티 피드**: 
  - Firestore 기반의 실시간 데이터 동기화.
  - 본인 글에 대한 수정/삭제 기능 제공.
  - 미인증 사용자를 위한 '인증 메일 재발송' 버튼 UI 제공.
