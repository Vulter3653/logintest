# 실시간 커뮤니티 피드 & 인증 시스템 블루프린트

## 개요
이 프로젝트는 실시간으로 소통할 수 있는 **글로벌 댓글 피드**를 메인 서비스로 제공합니다. 사용자는 로그인 없이도 실시간 메시지를 읽을 수 있으며, 인증 후에는 메시지 작성 및 자신의 댓글을 직접 관리(수정/삭제)할 수 있습니다.

## 주요 기능 및 디자인
- **메인 커뮤니티 피드 (`<comments-section>`)**:
  - **첫 화면 제공**: 사이트 접속 시 즉각적인 피드 노출.
  - **개선된 댓글 관리 UI**: 수정 시 **[저장]**, **[취소]** 버튼이 기존 버튼 위치에 나타나도록 최적화하여 직관적인 UX 제공.
  - **실시간 동기화**: Firestore의 실시간 스냅샷을 통한 즉각적인 데이터 반영.
- **모달 인증 시스템 (`<login-screen>`)**:
  - 필요할 때만 나타나는 세련된 팝업 방식 UI.
  - 이메일 로그인 및 구글 소셜 로그인 지원.
- **현대적인 UI/UX**:
  - **인라인 에디터**: 페이지 이동 없는 즉각적인 텍스트 수정 환경.
  - **반응형 디자인**: 모바일 및 데스크톱 모두에 최적화된 레이아웃.
