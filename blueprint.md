# 실시간 커뮤니티 피드 & 인증 시스템 블루프린트

## 개요
이 프로젝트는 실시간으로 소통할 수 있는 **글로벌 댓글 피드**를 메인 서비스로 제공합니다. 사용자는 로그인 없이도 실시간 메시지를 읽을 수 있으며, 인증 후에는 메시지 작성 및 자신의 댓글을 직접 관리(수정/삭제)할 수 있습니다.

## 주요 기능 및 디자인
- **메인 커뮤니티 피드 (`<comments-section>`)**:
  - **첫 화면 제공**: 사이트 접속 시 가장 먼저 실시간 댓글 목록을 보여줌으로써 즉각적인 사용자 참여 유도.
  - **실시간 동기화**: Firestore의 실시간 스냅샷을 통해 새로고침 없는 데이터 업데이트.
  - **댓글 관리**: 본인이 작성한 댓글에 대해 인라인 **수정** 및 **삭제** 버튼 제공.
- **모달 인증 시스템 (`<login-screen>`)**:
  - **팝업 방식 UI**: 서비스 이용 중 필요할 때만 나타나는 세련된 모달형 로그인 창.
  - **멀티 인증**: 이메일/비밀번호 방식과 구글 소셜 로그인 지원.
- **현대적인 UI/UX**:
  - **인라인 에디터**: 댓글 수정 시 별도 페이지 이동 없이 즉시 텍스트 영역 활성화.
  - **상단 고정 입력창**: 긴 피드를 읽으면서도 언제든 글을 작성할 수 있도록 입력 카드 고정.
  - **사용자 식별**: 본인 댓글의 시각적 강조 ('나' 배지 및 색상 차별화).
