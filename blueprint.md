# 로그인 화면 구현 블루프린트

## 개요
이 프로젝트는 사용자 인증을 위한 세련되고 현대적인 로그인 화면을 제공합니다. 프레임워크 없이 웹 표준 기술(Web Components, Modern CSS, ES Modules)과 Firebase를 사용하여 구현됩니다.

## 주요 기능 및 디자인
- **웹 컴포넌트 (`<login-screen>`)**: 독립적이고 재사용 가능한 로그인 폼 구성.
- **Firebase Authentication 통합**:
  - 실제 이메일/비밀번호 로그인 기능 구현 (`signInWithEmailAndPassword`).
  - 로그인 성공/실패에 따른 실시간 피드백 (에러 메시지 처리).
  - Firebase SDK v10 사용 및 모듈형 구성 (`firebase-config.js`).
- **현대적인 UI/UX**:
  - OKLCH 색상 체계를 사용한 생동감 있는 그라데이션 및 강조 색상.
  - 깊이감 있는 그림자(Drop Shadows)와 매끄러운 트랜지션.
  - 반응형 디자인 (컨테이너 쿼리 및 뷰포트 대응).
- **상호작용**:
  - 입력 필드 포커스 효과 및 버튼 글로우(Glow) 효과.
  - 로그인 진행 상태 표시 (버튼 비활성화 및 텍스트 변경).
