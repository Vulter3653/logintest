# 로그인 화면 및 실시간 커뮤니티 블루프린트

## 개요
이 프로젝트는 사용자 인증 기능을 포함한 **실시간 글로벌 댓글 피드** 서비스를 제공합니다. 이메일/비밀번호 인증과 구글 소셜 로그인을 지원하며, 모든 사용자가 실시간으로 소통할 수 있는 현대적인 웹 애플리케이션입니다.

## 주요 기능 및 디자인
- **Firebase Authentication 통합**:
  - **이메일/비밀번호**: 기존 방식의 로그인 및 회원가입.
  - **구글 로그인 (Google OAuth)**: `signInWithPopup`을 이용한 간편 소셜 로그인.
  - **도메인 보안**: `auth/unauthorized-domain` 에러 해결을 위해 승인된 도메인 관리 필수.
- **실시간 글로벌 피드 (`<comments-section>`)**:
  - **Firestore 연동**: 모든 사용자가 하나의 게시판을 실시간으로 공유.
  - **사용자 식별**: 본인 댓글 강조 및 실시간 데이터 동기화.
- **현대적인 UI/UX**:
  - **구글 버튼**: 브랜드 가이드 준수 디자인.
  - **다크 테마**: OKLCH 기반의 세련된 UI.

## 트러블슈팅 (Troubleshooting)
### 구글 로그인 시 `auth/unauthorized-domain` 오류 발생 시
- **원인**: 배포된 도메인이 Firebase 승인 목록에 없음.
- **해결**: Firebase 콘솔 > Authentication > Settings > Authorized domains에 해당 도메인(예: `logintest-ech.pages.dev`) 추가.
