# 로그인 화면 및 실시간 커뮤니티 블루프린트

## 개요
이 프로젝트는 사용자 인증 기능을 포함한 **실시간 글로벌 댓글 피드** 서비스를 제공합니다. 모든 로그인 사용자가 하나의 게시판을 공유하며 실시간으로 소통할 수 있는 현대적인 웹 애플리케이션입니다.

## 주요 기능 및 디자인
- **글로벌 실시간 피드 (`<comments-section>`)**:
  - **공유 게시판**: 모든 사용자가 하나의 Firestore 컬렉션을 공유하여 실시간으로 메시지를 주고받습니다.
  - **실시간 업데이트**: `onSnapshot` 기능을 통해 새로고침 없이 타 사용자의 글을 즉시 확인 가능합니다.
  - **사용자 식별**: 본인이 작성한 글은 특별한 스타일과 '나' 배지로 강조되어 가독성을 높였습니다.
- **Firebase Authentication 통합**:
  - 이메일/비밀번호를 이용한 안전한 인증 시스템.
  - 로그인 상태에 따른 자동 화면 전환 (로그인 폼 <-> 커뮤니티 피드).
- **현대적인 UI/UX**:
  - **다크 모드 디자인**: OKLCH 색상 체계를 적용한 눈이 편안한 다크 테마.
  - **반응형 최적화**: 모바일에서도 모든 텍스트와 버튼이 적절하게 배치됩니다.
  - **상호작용**: 메시지 전송 시 부드러운 애니메이션 효과.
