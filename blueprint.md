# 로그인 화면 구현 블루프린트

## 개요
이 프로젝트는 사용자 인증을 위한 세련되고 현대적인 로그인/회원가입/비밀번호 찾기 화면과 로그인 후 이용 가능한 **실시간 댓글 커뮤니티**를 제공합니다. 프레임워크 없이 웹 표준 기술(Web Components, Modern CSS, ES Modules)과 Firebase를 사용하여 구현됩니다.

## 주요 기능 및 디자인
- **웹 컴포넌트 (`<login-screen>`, `<comments-section>`)**: 독립적이고 재사용 가능한 인증 및 커뮤니티 구성 요소.
- **Firebase 통합 (공식 문서 기반)**:
  - **인증 (Auth)**: 로그인, 회원가입, 로그아웃, 비밀번호 재설정 기능.
  - **데이터베이스 (Firestore)**: 댓글 데이터 저장 및 `onSnapshot`을 통한 실시간 목록 동기화.
  - **상태 관리**: `onAuthStateChanged`를 사용하여 로그인 시 댓글 게시판으로 자동 전환.
- **현대적인 UI/UX**:
  - **게시판 레이아웃**: 카드 형태의 댓글 목록 및 텍스트 영역.
  - **반응형 최적화**: 모바일에서도 쾌적하게 댓글을 읽고 쓸 수 있는 유연한 레이아웃.
  - 깊이감 있는 그림자 효과 및 매끄러운 화면 전환 애니메이션.
- **상호작용 및 보안**:
  - 댓글 등록 시 로딩 상태 표시 및 자동 텍스트 비우기.
  - `escapeHTML` 처리를 통한 보안 강화.
